### **外星人入侵（Alien Invasion）README**

#### 一、项目概述

本项目是基于 Python + Pygame 开发的经典横向射击游戏，核心玩法为玩家操控飞船抵御外星人群体入侵，通过精准射击获取分数。项目在基础游戏逻辑上，额外实现了 历史最高分持久化记录 与 外星人子弹反击系统 两大特色功能，提升游戏挑战性与可玩性。

#### 二、核心功能模块

* ##### 基础游戏功能

1. 飞船操控：左右方向键控制移动，空格键发射子弹

2\. 外星人生成：自动批量生成外星人，按规律横向 / 纵向移动

3\. 碰撞检测：子弹击中外星人、外星人撞击飞船 / 屏幕底部的判定

4\. 游戏状态管理：生命值消耗、游戏暂停 / 重启 / 结束逻辑

* ##### 扩展实现功能

###### 1\. 历史最高分记录

\- 游戏退出时自动将当前最高分写入本地文件（high\_score.txt）

\- 游戏启动时加载历史最高分，实时对比并更新显示

###### 2\. 外星子弹系统

\- 外星人随机间隔发射子弹（通过 ALIEN\_FIRE\_EVENT 事件触发）

\- 外星子弹碰撞检测（击中外船扣血，超出屏幕自动销毁）

#### 三、环境准备与运行步骤

###### 1\. 依赖环境

Python 版本：3.10.0及以上

第三方库：Pygame（负责游戏图形渲染、事件监听、音效播放）

###### 2\. 安装依赖

打开终端 / 命令提示符，执行以下命令安装 Pygame：

\# 通用安装命令

pip install pygame

\# 若存在多个Python版本，指定Python3安装

pip3 install pygame

#### 四、项目文件结构说明

alien\_invasion.py（ 主程序文件）：游戏入口，包含游戏主循环（事件监听、画面渲染、元素更新）、初始化核心对象

settings.py（ 配置文件）：存储游戏全局设置：屏幕尺寸（1200x800）、飞船 / 子弹 / 外星人速度、颜色参数等

ship.py（ 飞船类（Ship））：初始化飞船图像 / 位置、移动逻辑（update()）、屏幕绘制（blitme()）

alien.py	（ 外星人类（Alien））：初始化外星人图像 / 位置、移动边界判定、群体生成逻辑

bullet.py（ 玩家子弹类（Bullet））：子弹生成、移动、碰撞检测、销毁逻辑

alien\_bullet.py（自定义外星子弹类（AlienBullet））：外星子弹生成、移动、碰撞判定

game\_stats.py（ 游戏统计类（GameStats））：管理得分、生命值、游戏状态，实现高分加载 / 保存

images（ 资源文件夹	存放游戏图像资源）：飞船（ship.bmp）、外星人（alien.bmp）、子弹（bullet.bmp）等

high\_score.txt（ 数据文件）：自动生成，存储历史最高分（纯文本格式，便于手动查看 / 修改）

#### 五、游戏操作指南

###### 操作按键 / 行为	        功能效果

左方向键（←）	                控制飞船向左移动

右方向键（→）	                控制飞船向右移动

空格键（Space）	                发射玩家子弹（单次发射 1 颗，有发射间隔限制）

Q 键	                                        快速退出游戏（退出时自动保存当前最高分）

鼠标点击（游戏结束后）	重启游戏（重置生命值、得分，保留历史最高分）
